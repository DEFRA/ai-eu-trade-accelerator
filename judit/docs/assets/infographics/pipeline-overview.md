# Pipeline overview (infographic spec)

Canonical source for infographic and slide design.

## Title

From sources to structured legal analysis

## Stages

Source intake → No model  
Proposition extraction → Local model (controlled)  
Proposition inventory & pairing → No model  
Proposition completeness → No model  
Source categorisation → No model  
Divergence classification → Optional frontier model  
Narrative output → Template-based (no model)

## Legend

- No model → deterministic processing
- Local model → controlled, cached
- Frontier model → optional, traceable
- Template-based → no generative AI

## Notes

- Most of the pipeline is deterministic
- Model usage is explicit and limited
- All outputs are traceable to source text
